This package provides interface to represent space objects such as stars or planets. Their geometry is represented by a {@link org.orekit.bodies.BodyShape} and other informations such as position and mass are represented by a {@link org.orekit.bodies.ThirdBody}.

Several implementations are provided by OREKIT : 

	- {@link org.orekit.bodies.OneAxisEllipsoid}, which represents the ellipsoid shape of space rotating bodies.
	- {@link org.orekit.forces.Moon} and {@link org.orekit.forces.Sun}, which provide position at a specific date and other informations such as the radius.
